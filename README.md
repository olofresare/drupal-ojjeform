drupal-ojjeform
========

Style form compontents with custom graphics without breaking event listeners. Same as ojjeform.js, just implemented as a drupal 7 module.

### Html example

### Javascript example 
```javascript
var $questions_form = $('#questions-form');

/* Style form components with jquery plugin */
if ($questions_form.length > 0) {
	var formOptions = {
		types: {
			checkbox: true,
			radio: true,
			select: true
		}
	};
	$.ojjeform($questions_form, formOptions);
}
```
